# NavList
Helper component for rendering Navigation and menu item lists.

<br/>

## Use
```js
  import { NavList } from '@keg-hub/keg-components'
```

### External Toggle Navigation
Section toggle is managed external to the NavList component. If using multiple item sections, the root toggled prop will be ignored. In this case, each section should contain its own toggle property.

**Example**
<Preview>
  <Story name="ExternalToggle">
    <ExternalToggle />
  </Story>
</Preview>

**Code**
```js
export const ExternalToggle = () => {
  const [toggled, setToggled] = useState(true)
  return (
    <NavList
      items={{
        external: {
          type: 'ExternalToggle',
          group: 'External Toggle',
          Icon: ChevronDown,
          items: [
            {title: `Nav Item 1`, uuid: 1},
            {title: `Nav Item 2`, uuid: 2},
            {title: `Nav Item 3`, uuid: 3}
          ]
        }
      }}
      toggled={toggled}
      onHeaderPress={() => setToggled(!toggled)}
    />
  )
}
```


### Top Level Only
If only passing top level items, the items prop accepts an **array** instead of an **object**.

**Example**
<Preview>
  <Story name="TopLevelOnly">
    <TopLevelOnly />
  </Story>
</Preview>

**Code**
```js
export const TopLevel = () => {
  return (
    <NavList
      items={[
        {title: `Item 1`, uuid: 1},
        {title: `Item 2`, uuid: 2},
        {title: `Item 3`, uuid: 3}
      ]}
    />
  )
}
```
