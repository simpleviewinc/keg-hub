# Modal
Base component for displaying content in a modal. Does **NOT** use the `react-naive` modal component.

## Examples

### Code
```js
  export const MyModal = (props) => {
    const [isOpen, setIsOpen] = useState(false)
    return (
      <View>
        <Modal
          visible={isOpen}
          onBackdropTouch={() => setIsOpen(false)}
        >
          {props.children}
          <Button
            onClick={() => setIsOpen(false)}
            content={'Close Modal'}
          />
        </Modal>
        <Button
          onClick={() => setIsOpen(true)}
          content={'Close Modal'}
        />
      <View>
    )
  }
```
