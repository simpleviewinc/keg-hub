  Use this hook to toggle between two styles based on the <b>active</b> state of the component. This hook should be combined with either the <b>withTheme HOC</b> or <b>useTheme hook</b> to get access to the theme.

### Example 
```js
import React from 'react'
import { useThemeActive, useTheme } from '@keg-hub/re-theme'

export const ActiveThemeButton = props => {
  const { button } = useTheme()
  const [ ref, theme ] = useThemeActive(button.default, button.active)

  return (
    <button ref={ ref } style={ theme } >
      { props.children }
    </button>
  )
}
```
